1.3.1 A benefit of Visual Studio Code is the power it brings to rapid code development. One of the drawbacks however, is that it takes time to learn to access this power. Efficiency of development could be greatly improved by spending more time learning the features of Visual Studio Code.
1.3.2 Productivity could be improved by learning more of the power-user features of Visual Studio Code. Also, extra team-members would enable task to be split across the team.
1.3.3 Next time I develop a website, I will probably spend more time designing the layout of the pages, and maybe spend time learning the features of the IDE/editor before I start to code.
1.3.4 Digital solutions can be tested by running the code regularly while in the development stage. After development is complete the site can be reviewed and checked for errors.
